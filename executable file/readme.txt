运行说明：
1.双击start.bat运行jar程序，由于在bat中设置了虚拟机的最大内存因此，若直接双击jar文件可能无法正常运行程序
2.目录下的data目录和test目录为必须目录若删除则会出错，因为程序默认会读取这里面的train等文件，当然也可以在程序界面中指定
  自定义的train和test等文件
3.程序运行后会弹出一个黑窗口用于显示当前处理的数据信息，若想终止程序执行请在黑窗口中按“Ctrl+C”来终止
4.程序的最终评分结果会输出到data下的result.txt中，每次重新运行前可以删除result文件，程序运行结束后会自动生成新的result文件
5.UI程序中没有涉及对测试的展现，若想查看测试程序请打开源程序，其中有个RecommendReport，运行可以得到统计信息（注，prePrepare可以自动生成测试元数据test和对比数据resl）
